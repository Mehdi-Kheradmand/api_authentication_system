Project Title: Django API Authentication System

Overview:

The Django API Authentication System is a project designed for showcasing proficiency in building secure and efficient authentication systems using Django

particularly focusing on the implementation of APIs. This project is purely intended as a demonstration piece for inclusion in a resume portfolio, not for commercial use or sale.

Features:

  1. API Authentication: The system implements various authentication methods, including username/password, token-based authentication, and OTP-based authentication.
  2. Token Management: Users are provided with tokens upon successful authentication, which are then used for subsequent API requests to maintain session state.
  3. OTP Verification: The project includes OTP verification as an additional layer of security, ensuring secure user authentication.
  4. User Management: Basic CRUD (Create, Read, Update, Delete) operations for user accounts are implemented through API endpoints.
  5. Session Management: The system efficiently manages user sessions and ensures proper session handling to maintain security and integrity.
